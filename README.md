# fre-miniapp

基于 fre 的小程序引擎，没什么好说的了，先开源吧，之后搞点别的

### Run

1. 环境

window11（window10 需要手动女装 webview2）

2. 先 build

```shell
yarn build
yarn build:wxml
```
3. 跑

```shell
yarn start
```
4. 预览
